# The Milk Store
A store to buy all kinds of milk :)

&nbsp;

### This application uses a MongoDB database
Database name: milkDb
&nbsp;

Collection(s) in the database: products

&nbsp;

## How to run the application?
Step 1 - Start frontend:
```
cd client
npm i
npm run dev
```
&nbsp;

Step 2 - Start backend:
```
cd server
npm i
npm start
```
